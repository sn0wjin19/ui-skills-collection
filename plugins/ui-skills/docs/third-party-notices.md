# Third-Party Notices

This plugin is mostly a synthesis. Source material was summarized and transformed into compact skill guidance.

The `ui-craft` reference folder now vendors selected Anthropic design skill files and theme examples under `skills/ui-craft/references/design-examples/anthropic/` so they can be loaded directly as concrete examples. Those files retain upstream license references in the same folder. No third-party executable code or datasets are vendored.

## Referenced And Synthesized

These sources informed the combined skill language and review priorities:

- Impeccable: https://github.com/pbakaus/impeccable, Apache-2.0
- Anthropic Skills: https://github.com/anthropics/skills, Apache-2.0 for vendored design skill references
- 0xdesign design-plugin: https://github.com/0xdesign/design-plugin
- User Interface Wiki: https://github.com/raphaelsalaja/userinterface-wiki, MIT
- Designer Skills: https://github.com/Owl-Listener/designer-skills, MIT
- Make Interfaces Feel Better: https://github.com/jakubkrehel/make-interfaces-feel-better, MIT

## Reviewed And Removed

- UI/UX Pro Max: https://github.com/nextlevelbuilder/ui-ux-pro-max-skill, MIT
- Taste Skill: https://github.com/Leonxlnx/taste-skill, MIT
- ui-skills: https://github.com/ibelick/ui-skills, MIT

`ui-design-brain` was reviewed from the located `thierry1804/duosite` repository, but no reusable skill file was found in that source.
